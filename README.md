# EduMeet

**Technologies Used**

- React
- Node + Express
- WebRTC
- Socket.io
- [Simple-peer](https://github.com/feross/simple-peer)

## Features

- Join a room
- Video Streaming
- Text chat
- Mute Video/Audio
- Screen Sharing

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/i-punithgowda/EduMeet`

### Setup

**Client**

> Move to client folder, update and install this package

<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm start
  </code>
</pre>

**Server**

> Move to server folder, update and install this package

<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm run dev
  </code>
</pre>
